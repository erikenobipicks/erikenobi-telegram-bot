# Landing Instagram

Esta carpeta contiene la versión `link in bio` de Erikenobi Picks.

Objetivo:

- usarla como enlace de Instagram
- llevar tráfico frío al canal free o a Telegram
- ofrecer accesos rápidos a los planes premium

Archivos:

- `index.html`: página principal
- `styles.css`: diseño mobile-first
- `script.js`: animaciones suaves
- `social-preview.svg`: preview social

## Publicación rápida

Si usas los archivos de este repo:

- `netlify.toml` publica `landing-instagram`
- `vercel.json` sirve `landing-instagram` en la raíz del dominio
